# Factor

A Stack-oriented language written in Rust

## Builtin Functions

- `add` (+),
- `sub` (-)
- `mul` (\*)
- `div` (/)
- `mod` (%)
- `pow` (\*\*)
- `dot` (.) prints out top of stack
- `dump` prints out whole stack
- `abs`
- `neg`
- `dup`
- `drop`
    - a b --> a
- `swap`
- `over`
    - a b --> a b a
- `rot` rotates top three stack elements
    - a b c --> b c a
- `clear` clears the stack
- `exit`
